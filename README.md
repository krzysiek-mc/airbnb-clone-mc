# AIRBNB CLONE

App is clone of AirBnb

## DEPENDENCIES

* node 16
* yarn

## HOW TO DEVELOP

1. Install dependencies

```bash
$ yarn install
Install all dependencies
```

1. DEBUGGING TOOLS

* React Native Debugger (<https://docs.expo.dev/workflow/debugging/#react-native-debugger>)

1. Start dev server

```bash
$ yarn start
Follow instructions on screen
```

## HOW TO TEST

1. Linters

```bash
$ yarn linter-prettier
Run prettier check
$ yarn linter-eslint
Run eslint check
$ yarn linters
Run both checks
```

1. Tools

```bash
$ yarn tool-expo-doctor
Run expo doctor
```
