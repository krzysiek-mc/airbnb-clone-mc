# AIRBNB CLONE

App is clone of AirBnb

## DEPENDENCIES

* node 16
* yarn

## HOW TO DEVELOP

1. Install dependencies

```bash
yarn install
```

1. Start dev server (follow instructions on screen)

```bash
yarn start
```

## HOW TO TEST

1. Linters (prettier, eslint)

```bash
yarn linter
```

1. Expo Doctor

```bash
yarn expo-doctor
```
